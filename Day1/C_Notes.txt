**Variables **
    -> syntax:- keyword identifier = value;
                Ex. int a=10;           -> this is called variable initialization
                Ex. char c='A';         -> this is called variable declaration
    -> variable is user defined memory allocation used to store data like numbers, strings etc.
    -> Variable is a simply name of storage location which is used to identify our data.
    -> We have to follow some rules to declare the variable names 
                - we can't use any repeated name (identifier)
                - we can't use a keyward as a name (keywords: int, float, char, etc.)
                - we can't use white-spaces while writting identifiers for concatenation purpose we use underscore ("_")
                - we can use alphabets underscore, character and number in our variable name; but we can't start from number

**Datatypes**
    -> is a concept which is used to specify the type of data
    -> Datatypes used to specify the size of data

    There are 4 Types of Data Types: 
    1. Basic        - Int (4 bytes), char(1 bytes), float(4 bytes), double(8 bytes)
    2. Derived      - Array, Pointer, structure, union
    3. Enumeration  - enum (mostly used for dates or days like that)
    4. Void         - void
